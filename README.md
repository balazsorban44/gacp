## Installation

`npm i -g gacp`

## Usage

`gacp`

`gacp --help`

`gacp --no-push`

`gacp --emoji emoji`

## Configuration File

You can configure gacp via:

- A `gacp` property in `package.json`.
- A `.gacprc` file in JSON, YAML or CommonJS with or without extensions `.json`, `.yaml`, `.yml`, `.js`.
- A `gacp.config.js` file in CommonJS.

### Basic Configuration

With default configuration.

```json
{
  "push": true,
  "emoji": "code"
}
```
